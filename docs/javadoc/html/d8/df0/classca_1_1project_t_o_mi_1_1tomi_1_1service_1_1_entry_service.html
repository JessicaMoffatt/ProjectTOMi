◆ updateEntry()

Entry ca.projectTOMi.tomi.service.EntryService.updateEntry ( final Long  id,
final Entry  updatedEntry 
)

Updates the Entry with the provided id with the provided attributes.

If the updated Entry has a Status of APPROVED or SUBMITTED, an IllegalEntryStateException will be raised as these statuses do not allow any modification of the contents of an Entry object.

If the updated Entry has a Status of LOGGING, the existing Entry will be updated to contain the updated attributes.

If the updated Entry has a Status of REJECTED, a new Entry object will be created with identical content to the rejected Entry and it will be saved with an Active status of false. The existing Entry Status will be set to LOGGING.

Parameters
idthe unique identifier for the Entry to update.
updatedEntryEntry object containing the updated attributes.
Returns
Entry containing the updated attributes.
